# Telecom Customer Churn Analysis & Prediction

##  Project Overview
Customer churn is a major challenge in the telecommunications industry, as losing customers impacts revenue.

This project analyzes customer data to:
- Identify key factors driving churn
- Build predictive models to estimate churn risk
- Provide insights using data visualization

---

##  Business Problem
Telecom companies need to understand why customers leave and which customers are at risk.

This project answers:
- What factors drive churn?
- Can we predict customer churn?

---

##  Dataset
The dataset contains 7,043 customer records and 21 variables including:

- Demographics (gender, dependents)
- Services (internet, tech support, streaming)
- Account info (tenure, contract, payment method)
- Target: Churn (Yes/No)

Source: IBM Telco Customer Churn Dataset

---

##  Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn)
- JMP (Modeling)
- Tableau (Dashboard)
- GitHub

---

##  Data Analysis Process
- Cleaned missing values and data types
- Performed exploratory data analysis (EDA)
- Built models:
  - Logistic Regression
  - Decision Tree
  - Bootstrap Forest
  - Boosted Tree
- Evaluated performance using R² and accuracy

---

##  Modeling Results
- Logistic Regression: R² ≈ 0.27  
- Decision Tree: R² ≈ 0.29  
- Bootstrap Forest: R² ≈ 0.41  
- Boosted Tree: R² ≈ 0.42 (Best)

---

##  Key Insights
- Month-to-month contracts have higher churn
- Customers with low tenure are more likely to leave
- Lack of tech support increases churn
- Higher monthly charges increase churn risk
